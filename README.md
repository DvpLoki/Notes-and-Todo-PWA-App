## Features

- Installable PWA app ( can install in any device Desktop or Mobile)
- Offline React SPA
- Rich Dynamic Markdown text (no need previews for markdown)
- todo list ( soon )

### docker command

`docker build -f Dockerfile.dev -t notesapp:v1 .`
